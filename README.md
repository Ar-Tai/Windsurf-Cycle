## 📦 安装方式

1. 下载最新的 `windsurf-cycle-x.x.x.vsix` 文件
2. 打开 **Windsurf** 或 **VS Code**
3. 按 `Ctrl+Shift+P` 打开命令面板
4. 输入 `Extensions: Install from VSIX`
5. 选择下载的 `.vsix` 文件
6. 重新加载窗口

## 🚀 快速开始

### 1️⃣ 激活

首次使用需要激活许可证：
- 打开侧边栏的 **Windsurf Cycle** 面板
- 输入卡密进行激活
- 激活成功后即可使用

### 2️⃣ 开始使用

- AI 完成任务后会自动询问是否继续
- 选择「继续」可无缝进行下一个任务
- 选择「结束」停止当前对话

## ❓ 常见问题

### Q: 如何获取设备ID？

在激活界面点击设备ID即可复制。

### Q: 卡密可以换设备吗？

可以，先在旧设备上解绑，再在新设备上激活。

### Q: 网络不好能用吗？

- **永久卡密**: 完全离线可用
- **限时卡密**: 支持24小时离线宽限期

### Q: 如何续期？

在面板中输入新的续期卡密即可，天数会累加。
